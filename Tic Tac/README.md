# Tic Tac Toe Game

A simple web-based Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

- Two-player game (X and O)
- Win detection
- Draw detection
- Reset functionality
- Responsive design

## How to Play

1. Open `index.html` in your web browser
2. Players take turns clicking on empty cells
3. First to get three in a row (horizontally, vertically, or diagonally) wins
4. Click "Reset Game" to start over

## Files

- `index.html`: The main HTML structure
- `style.css`: Styling for the game
- `script.js`: Game logic and interactivity</content>
